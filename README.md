# Delta Huffman X-Ray Compression

Repository ini berisi implementasi kompresi citra X-Ray menggunakan algoritma
**Delta Encoding + Huffman Coding** yang dijalankan menggunakan **Kaggle Notebook**.

⚠️ Notebook **tidak dijalankan secara lokal**. Seluruh proses eksperimen dilakukan di environment Kaggle.

---

## Environment
- Platform      : Kaggle Notebook (Cloud)
- Bahasa        : Python
- Format File   : `.ipynb`

---

## How to Run on Kaggle (Step-by-Step)

### 1. Buat Notebook Baru
1. Buka https://www.kaggle.com
2. Login ke akun Kaggle
3. Klik **Create → New Notebook**
4. Pilih **Notebook (Python)**

---

### 2. Import Notebook dari Repository
1. Di halaman Notebook Kaggle, klik menu **File**
2. Pilih **Import Notebook**
3. Upload file:
4. Setelah berhasil di-import, seluruh cell notebook akan muncul di editor Kaggle

---

### 3. Menjalankan Program
1. Pastikan notebook **sudah dalam keadaan terbuka**
2. Klik menu **Run → Run All**
3. Kaggle akan menjalankan cell dari atas ke bawah secara berurutan

---

### 4. Melihat Output
- Output setiap proses **akan muncul tepat di bawah masing-masing cell**
- Grafik, tabel, dan hasil analisis ditampilkan langsung setelah cell dieksekusi
- Jika terdapat beberapa tahap eksperimen, setiap tahap memiliki output tersendiri

---

### 5. Catatan Penting
- Notebook harus dijalankan secara berurutan mulai dari cell pertama
- Jangan melewatkan cell apa pun karena setiap cell saling bergantung
- Jika terjadi error, gunakan **Restart & Run All**
